Anker Soundcore Mini does not react on volume control from Windows

Recently I had an issue with Anker Soundcore Mini Bluetooth speaker. Windows volume level was not working with it. Only volume level 0 worked. Any other level just unmuted the speaker.

When changing the volume from the anker it worked and volume on Windows was also changed.

Here is how I was able to fix it:

  • Change the value of HKLM\SYSTEM\ControlSet001\Control\Bluetooth\Audio\AVRCP\CT\DisableAbsoluteVolume to 1
  • Restart Windows

Side effect is that the volume control on the anker does not synchronize with Windows.
